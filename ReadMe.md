# Feladat

Egy interaktív önéletrajz létrehozása, munkákkal kiegészítve.

## Kivitelezés

Álló formátumú önéletrajz, munkák bemutatásával, rövid ismertetővel.
A méhecskére kattintva sötét verzióra tudunk váltani.
Nyelvet is tudunk váltani jelenleg magyar és angol áll rendelkezésre.
A munkák között videó is található. Illetve egy halk zene is tásul az oldalhoz.
